
# Projeto Linux

Projeto Linux desenvolvido no contexto do Programa de Bolsas.

## Tecnologias/ferramentas utilizadas:
- VirtualBox – para criação da máquina virtual
- Maquina Virtual (Oracle VirtualBox)
- Ubuntu Server (Imagem ISO)
- Nginx 
- HTML
- Bash 
- cURL 
- Cron
